# Custom Slash Commands

This directory holds project-level custom slash commands for the SaVi banking app.
Each `.md` file here becomes available as `/filename` inside Claude Code.

## Commands (built progressively)
- `add-route.md` → `/add-route` — scaffold a new Flask route with auth
- `banking-test.md` → `/banking-test` — run and summarize test results
- `security-audit.md` → `/security-audit` — check for OWASP issues in the codebase

Commands will be added in Step 3 of the learning roadmap.
